# Accompany

# Genel Özet
Accompany uygulaması Android ve IOS cihazlarda, kolay kullanılabilen bir arayüz ile şirket içerisindeki iletişimi artırmayı hedefleyen bir mobil uygulamadır.

Kayıt/Giriş işlemleri
Haber görüntüleme
